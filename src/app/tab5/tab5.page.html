Tentang Aplikasi

Lughaty Digital

Media Komunikasi Dakwah Digital

Tentang Aplikasi

Pengembangan Bahan Ajar Bahasa Arab Interaktif sebagai Media Komunikasi Dakwah Digital. Aplikasi ini dirancang untuk membantu pengguna belajar bahasa Arab dengan mudah dan menyenangkan.

Teknologi yang Digunakan

Ionic 7 + Angular 17

Framework utama untuk membangun UI

Supabase

Database untuk menyimpan data pengguna

Capacitor 5

Wrapper untuk Android & iOS

Axios & Swiper

HTTP client & slider interaktif

Fitur

Al-Istima - Latihan mendengar
Al-Kalam - Percakapan sehari-hari
Al-Qiroah - Kuis pemahaman
Kitabah - Latihan menulis

Dikembangkan Oleh

Mahasiswa Universitas Ibn Khaldun (UIKA) Bogor

Fakultas Agama Islam - Program Studi Pendidikan Bahasa Arab
